Содержит класс graph - обычный класс графа. У каждой вершины есть значений, хранимое в ней, а также список из ее родителей.
Помимо этого есть graph_operations, где находится функция поиска самого длинного пути, которая называется get_longest_path. В функцию подается указатель на одну вершину графа. Далее, используя алгоритм обхода в ширину, функция ищет самый длиный путь от нее до тупиковой вершины (той, у которой нет родителей). Возвращает функция пару значений: первое - длину самого длинного пути, второе - список из указателей на вершины, из которых состоит этот путь, в порядке прохода от найденой тупиковой вершины до заданой изначально.
